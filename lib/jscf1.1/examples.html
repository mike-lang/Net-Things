JavaScript Collections Framework 1.0

An example is worth a thousand words. Here is how you would use the framework.

Using ArrayList.

  1. Include ArrayList.js file.
  2. To create an arraylist say

        var list = new ArrayList(20);

        list.add("1");

        list.add("foo");

        list.add(new Object());

        list.add(new AnyObject());

        // iterate through the items

        for(itr = list.iterator(); itr.hasNext();){

            document.write(itr.next());

        }

        list.remove("foo");

        blah...blah...blah...

 

Using HashMap  

        var map= new HashMap

        map.put("1", "2");

        map.put("foo", "bar");

        map.put(new Object(), new AnotherObject());

        map.put(new AnyObject(), new AnyOtherObject());

        // iterate through the items

        for(itr = map.keySet().iterator(); itr.hasNext();){

            document.write(map.get(itr.next()));

        }

        map.remove("foo");

        blah...blah...blah...    

And similarly HashSet, LinkedHashSet, LinkedHashMap can be used. I hope this examples help. If not
